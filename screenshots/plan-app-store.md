## Elementos de Diseño Profesional Necesarios

### 1. Device Frame (Marco del iPhone)
- Usar el marco oficial del **iPhone 15 Pro Max**
- Incluir Dynamic Island
- Esquinas redondeadas perfectas
- Sombra sutil para profundidad

**Recursos recomendados:**
- Apple Design Resources (oficial)
- DeviceFrames.com
- MockuPhone.com

### 2. Barra de Estado Limpia
Para todas las capturas, ajustar:
- ⏰ Hora: **9:41 AM** (estándar de Apple)
- 🔋 Batería: 100%
- 📶 Señal: Completa
- 📡 WiFi: Activo

### 3. Backgrounds (Fondos)
**Opciones recomendadas:**

**Opción A - Degradados suaves:**
- Captura 1: Degradado azul oscuro → azul medio (#1a237e → #0d47a1)
- Captura 2: Degradado verde turquesa → verde claro (#00695c → #26a69a)
- Captura 3: Degradado azul cielo → blanco (#0277bd → #ffffff)
- Captura 4: Degradado verde → verde claro (#2e7d32 → #66bb6a)
- Captura 5: Degradado gris → gris claro (#424242 → #757575)

**Opción B - Colores sólidos con overlay:**
- Usar el color principal de la app (turquesa #26a69a) con overlay de 70% de opacidad

### 4. Tipografía
- **Fuente principal:** San Francisco (oficial de Apple)
- **Alternativa:** SF Pro Display (disponible en Apple Developer)
- **Tamaño título:** 80-100pt
- **Tamaño subtítulo:** 44-52pt
- **Color texto:** Blanco con sombra sutil para legibilidad
- **Alineación:** Centrado o alineado a la izquierda según el espacio

## Reglas de Oro de Apple (Checklist de Aprobación)

### ✅ Cumplimiento Obligatorio
- [ ] Dimensiones exactas: 1290 x 2796 px
- [ ] Sin mencionar precios o promociones ("Gratis", "Oferta")
- [ ] Sin referencias a competidores o plataformas (Android, etc.)
- [ ] Mostrar interfaz real de la app (no solo ilustraciones)
- [ ] Barra de estado limpia y consistente
- [ ] Sin contenido ofensivo o inapropiado
- [ ] Texto legible sin hacer zoom
- [ ] Perfil de color RGB o Display P3

### ⚠️ Recomendaciones Adicionales
- [ ] Máximo 10 capturas por idioma
- [ ] Orden coherente que cuente una historia
- [ ] Textos cortos y de acción (verbos imperativos)
- [ ] Alto contraste entre texto y fondo
- [ ] Evitar textos demasiado pequeños (<40pt)

## Flujo de Trabajo de Producción

### Opción 1: Herramientas Online (Rápido)
1. **AppScreens.com** - Automatizado, pago
2. **PlaceIt by Envato** - Templates prediseñados
3. **MockuPhone** - Gratuito, básico

### Opción 2: Diseño Manual (Control Total)
1. **Software:** Figma, Adobe Photoshop, Sketch
2. **Proceso:**
   - Crear canvas de 1290 x 2796 px
   - Insertar device frame del iPhone 15 Pro Max
   - Colocar screenshot dentro del frame
   - Añadir background con degradado
   - Insertar textos con tipografía San Francisco
   - Exportar como PNG de alta calidad

### Opción 3: Script Automatizado (Recomendado para este proyecto)
Usar herramientas de línea de comandos:
- **ImageMagick** para composición
- **Python PIL/Pillow** para procesamiento
- **Figma API** para automatización completa
